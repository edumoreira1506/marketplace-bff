# marketplace-bff

## Dependencies

```
NodeJS >= 16.14.0
Postgres >= 13
```

## Setup

Clone the project.

```console
$ git clone git@github.com:edumoreira1506/marketplace-bff.git
$ cd marketplace-bff
```

Copy the contents of the `.env.sample` to `.env` then change with the credentials of your local environment.

```console
$ cp .env.sample .env
```

Install all packages.

```console
$ npm i
```

For run server local.

```console
$ npm run dev
```

## Tests

For run tests.

```console
$ npm run test
```

## Linter

For run eslint.

```console
$ npm run lint
```
